# Mood2Meme

## 프로젝트 소개

Mood2Meme는 실시간 웹캠 영상에서 사용자의 포즈를 인식하고,
해당 포즈와 가장 잘 어울리는 짤(meme) 이미지를 추천하는
컴퓨터 비전 기반 프로젝트이다.

사용자는 브이(V) 포즈, 손하트, 화난 포즈 등의 다양한 포즈를 취할 수 있으며,
프로그램은 이를 분석하여 미리 준비된 짤 이미지 중 적절한 결과를 화면에 표시한다.

이 프로젝트는 OpenCV를 활용하여 구현할 예정이다.

---

## 개발 목표

- 실시간 웹캠 입력 처리
- 사람의 포즈 검출
- 포즈 분류
- 포즈에 맞는 짤 이미지 표시

---

## 사용 기술

- Python
- OpenCV

---

## 예상 기능

- 웹캠 기반 실시간 포즈 인식
- 여러 종류의 포즈 지원
- 포즈별 짤 이미지 표시
- 실시간 화면 표시

---

## 프로젝트 진행 예정 사항

1. 포즈 데이터 수집
2. 포즈 검출 구현
3. 짤 데이터셋 구성
4. 포즈와 짤 매칭 시스템 구현
5. 최종 테스트
